Using the terminal/shell

The shell is an interface for navigating and performing functions on files in your computer. In most cases, you will only need to use it to copy or delete files. Both the Rubin Science Platform and your installation of Anaconda Navigator (which will be covered in the following sectin on Python) will provide you with access to a version of this interface. In Anaconda it will be called “Anaconda/Conda Prompt”. On the Rubin Science Platform it will be called a terminal.

Remember that the Rubin Science Platform can only access files stored in its web-based directories. Your installation of Anaconda can access the files hosted locally on your computer.

It is quite likely that you will rarely need to use the shell. It is still useful to become familiar with it, as it can make moving, copying, and renaming large batches of files much easier and faster. An in-depth tutorial is available at Software Carpentry's "The Unix Shell" lesson. Briefer overviews are available at the links below.

  • "Learning the Shell" will help you get started with basic commands like changing directories and listing the files inside a directory.
  • explainshell.com is a useful tool that allows you to enter a command and gives you a description of what that command does.

Using SQL

SQL, or Structured Query Language, is a programming language used when accessing data that is stored in a type of structure called a relational database. Essentially, data is stored in tables sorted by rows and columns, similarly to a spreadsheet. The language enables the abilities to create, look up, modify, and relate data to one another.

It also enables fast sorting and filtering of large data sets, which are very common in the field of astronomy.
While there are many uses for SQL, you will currently only need it to access data from the different astronomical surveys, where we will be searching for merging galaxy pairs.

  • The SDSS SQL Tutorial will get you started using SQL queries to explore the vast amounts of data in the survey.
  • If you get stuck or would like a more thorough lesson on using SQL, the Software Carpentry tutorial on Using Databases and SQL is very helpful.

Basic Cosmology

There are a few pieces of information that are currently accepted as fact in the field of cosmology:

  1. The universe is expanding.
  2. The rate of this expansion is increasing.
    • This rate is known as the Hubble constant, or H0. Its value is approximately 70 km/s/Mpc, but there is currently contention about its exact value.

What are Right Ascension and Declination?

Right ascension and declination are two dimensions of the equatorial coordinate system, which is most commonly used to refer to the positions of astronomical objects. Its origin is Earth's center, which makes it a geocentric coordinate system. It is also a spherical coordinate system, rather than rectangular. Right ascension is the equivalent to longitude, and declination is the equivalent to latitude.

  • Right ascension is measured in hours, minutes, and seconds. There are 60 minutes in an hour and 60 seconds in a minute, the same way time is measured. These smaller units are also commonly called arcminutes and arcseconds, to differentiate them from units of time measurement. This means that a perfect circle is 24 hours. In other words, a single hour of right ascension is equivalent to 360°⁄24h= 15°. An arcminute is 1⁄60 of an hour, or 1⁄1440 of a circle. An arcsecond is 1⁄3600 of an hour, or 1⁄86400 of a circle.
    As an example, the moon is approximately 31 arcminutes in size, or a little more than 1⁄2 of an hour.
    Hours are typically denoted by the letter h, arcminutes by the symbol ', and arcseconds by the symbol ".
  • Declination is measured in the same way, but comprises the vertical axis of the celestial coordinate system: it is the celestial equivalent to latitude.
